# Dashboard Redesign

This project focused on transforming a cluttered analytics dashboard into a streamlined, intuitive interface.

## Key Features

- Real-time data updates with WebSocket integration
- Customizable widget layouts
- Dark mode support
- Export functionality for reports

## Technical Stack

Built with React and D3.js for complex data visualizations, integrated with a Node.js backend.
